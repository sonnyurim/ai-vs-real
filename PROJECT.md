# Find the AI — AI가 숨긴 사물 찾기 퀴즈

연암공과대학교에서 만든 퀴즈 게임. 실제 사진 속에 AI가 몰래 그려 넣은 사물을 찾아내는 감별 게임입니다.

---

## 컨셉

"이 사진에서 AI가 몰래 넣은 건 뭘까?"

실제 사진 위에 AI가 자연스럽게 그려 넣은 사물이 숨어 있습니다. 난이도에 따라 찾아야 할 사물이 1~3개로 늘어납니다. 플레이어는 이미지를 살펴보고, AI가 추가한 부분을 직접 클릭해서 찾아냅니다. 10문제를 풀며 자신의 감별 실력을 점수와 등급으로 확인하고, 다른 플레이어들과 순위를 겨룰 수 있습니다.

---

## 게임 흐름

### 1. 닉네임 입력

접속하면 바로 닉네임 입력창이 나옵니다. 랭킹에 올라갈 이름입니다. 2~10자 제한.

```
연암공과대학교

Find the AI
실제 사진 속에 AI가 몰래 넣은 사물을 찾아라!
당신의 눈은 AI를 잡아낼 수 있나요?

[ 닉네임 입력 ]
      퀴즈 시작

10문제 / 최대 150점
```

### 2. 퀴즈 (10문제)

**상단 진행 바**가 항상 보입니다.

```
1/10    ⏱ 10    0점    [찾음: 0/1]
━━━━━━━━━━━━━━━━━━━━━━━
```

- 왼쪽: 현재 문제 번호
- 가운데: 남은 시간 (초 단위 카운트다운)
- 오른쪽: 누적 점수
- 진행 막대: 문제를 풀수록 채워짐

**타이머:**

- 난이도별 제한 시간: **Easy=10초, Medium=15초, Hard=20초**
- 이미지가 표시되는 순간부터 카운트다운 시작
- 남은 시간이 3초 이하가 되면 숫자가 **빨간색**으로 바뀌며 깜빡임
- 시간이 0이 되면 미발견 객체 전부 **자동 오답 처리** → 정답 공개 화면으로 이동

---

#### 문제 화면

실제 사진 위에 AI가 생성한 사물이 난이도에 따라 1~3개 추가되어 있습니다. 플레이어는 이미지를 살펴보고 AI가 넣은 사물을 모두 찾아 클릭합니다.

```
이 사진에서 AI가 몰래 넣은 사물을 모두 찾으세요! (찾음: 0/2)

┌──────────────────────────────────┐
│                                  │
│         [합성된 이미지]            │
│                                  │
│   (클릭 가능 — 커서가 돋보기로     │
│    바뀌며 이미지 위를 탐색)        │
│                                  │
└──────────────────────────────────┘

  객체 1: ●●● (3회 남음)  💡 힌트
  객체 2: ●●● (3회 남음)  💡 힌트
```

**인터랙션 디테일:**

- 이미지 위에서 커서가 돋보기(🔍) 아이콘으로 변경
- 클릭하면 해당 위치에 원형 마커가 표시됨
- AI 사물의 **판정 영역**은 해당 사물을 감싸는 사각형 히트박스로 설정 (여유 마진 포함)
- 클릭 기회는 **객체당 3회 독립 부여** — 각 객체의 남은 클릭 횟수가 하단에 표시됨

```
객체 1: ●●● (3회 남음)  💡 힌트
객체 2: ●●● (3회 남음)  💡 힌트
```

- 3회 안에 해당 객체의 정답 영역을 클릭하면 그 객체 정답 처리 → 즉시 초록 하이라이트
- 3회 모두 빗나가면 해당 객체만 오답 처리 + 정답 공개, 나머지 객체 탐색 계속
- **모든 객체가 찾거나 찬스 소진 완료되면 NEXT 버튼 활성화**

---

#### 힌트 시스템

각 객체마다 힌트 1회 사용 가능합니다 (객체 ID에 귀속). 힌트를 쓰면 **해당 객체의 점수만 절반** (소수점 내림)으로 감소하며, 다른 객체의 점수에는 영향이 없습니다.

```
객체 1: ●●● (3회 남음)  [💡 힌트 보기]
객체 2: ●●● (3회 남음)  [💡 힌트 사용 (점수 ½)]  ← 사용 후 비활성화
```

**힌트 유형** (객체마다 다름):

- **영역 힌트**: 이미지 위에 넓은 원형 범위가 표시되어 해당 AI 사물이 있는 대략적 위치를 알려줌
- **텍스트 힌트**: "동물을 찾아보세요", "하늘을 자세히 보세요" 같은 방향 제시

힌트를 사용하면 해당 객체의 버튼이 비활성화됩니다.

---

### 3. 정답 공개

객체를 찾을 때마다 즉시 초록 하이라이트가 표시되고, 모든 객체가 완료되면 NEXT 버튼이 활성화됩니다.

#### 진행 중 — 객체 1개 찾은 경우 (Medium 문제)

```
이 사진에서 AI가 몰래 넣은 사물을 모두 찾으세요! (찾음: 1/2)

┌──────────────────────────────────┐
│                                  │
│         [합성된 이미지]            │
│                                  │
│     ✅ [고양이] 초록 하이라이트    │  ← 찾은 객체
│     ? [우산] 아직 탐색 중         │
│                                  │
└──────────────────────────────────┘

객체 1 [고양이]: ✅ 찾음 (+5점)
객체 2 [우산]:   ●●○ (1회 남음)  💡 힌트
```

#### NEXT 화면 — 모든 객체 완료 후

```
  결과: 객체 2/2 발견  (또는 1/2 발견 등)

┌──────────────────────────────────┐
│                                  │
│         [합성된 이미지]            │
│                                  │
│  ✅ [고양이] 정답 위치 + 클릭 마커 │
│  ✅ [우산] 정답 위치 + 클릭 마커   │  ← 오답인 경우 빨간 마커
│                                  │
└──────────────────────────────────┘

  객체 1 [고양이]: +5점 (힌트 사용)
  객체 2 [우산]:   +5점
  시간 보너스:     +3점
  합계:           +13점             [NEXT →]

다른 사람들의 정답률
┌──────────────────────────────────┐
│ 전체 정답  38%  ███████░░░░░░░░  │
│ 부분 정답  42%  ████████░░░░░░░  │
│ 오답       20%  ████░░░░░░░░░░░  │
└──────────────────────────────────┘
```

**공통 요소:**

- 찾은 객체는 즉시 초록 하이라이트 + 정답 마커 표시 (탐색 중 실시간)
- 이미지 위 "찾음: N/M" 진행 표시
- 찬스 소진된 객체: 자동으로 정답 위치 공개 (주황 테두리)
- NEXT 화면에서 모든 객체의 정답 위치 + 플레이어 클릭 위치 모두 표시
- 다른 플레이어들의 정답률 표시
- NEXT 버튼으로 다음 문제 이동 (모든 객체 완료 시에만 활성화)

전체 플레이어 순위를 확인합니다. 점수가 높을수록, 같은 점수면 힌트를 적게 쓸수록, 그래도 같으면 빨리 끝낼수록 높은 순위입니다.

```
│ ...                            │

```

## 점수 체계

### 기본 점수

난이도별 찾아야 할 객체 수가 다르며, 더 많이 찾을수록 점수가 올라갑니다.

| 문제 유형 | 정답 1개 | 정답 2개 | 정답 3개 |
|-----------|---------|---------|---------|
| Easy (객체 1개) | 10점 | — | — |
| Medium (객체 2개) | 5점 | 10점 | — |
| Hard (객체 3개) | 3점 | 6점 | 10점 (3·3·4 분배) |

> Hard 3개 분배: 객체 1번=3점, 객체 2번=3점, 객체 3번=4점

### 시간 보너스

**모든 객체를 찾은 경우에만** 적용됩니다. 마지막 객체를 찾은 시점의 남은 시간 비율 기준.

| 남은 시간 비율 | 보너스 |
|--------------|--------|
| 80% 이상 | +5점 |
| 50~79% | +3점 |
| 20~49% | +1점 |
| 20% 미만 | +0점 |

실제 초 기준:

| 난이도 | +5점 | +3점 | +1점 | +0점 |
|--------|------|------|------|------|
| Easy (10초) | 8~10초 | 5~7초 | 2~4초 | 0~1초 |
| Medium (15초) | 12~15초 | 8~11초 | 3~7초 | 0~2초 |
| Hard (20초) | 16~20초 | 10~15초 | 4~9초 | 0~3초 |

- **최대 점수: 문제당 15점 (기본 10 + 보너스 5) × 10문제 = 150점**

### 힌트 사용 시 점수

| 상황 | 점수 |
|------|------|
| 힌트 사용 객체 정답 | 해당 객체 점수 × 0.5 (소수점 내림) |
| 힌트 미사용 객체 정답 | 정상 점수 |
| 시간 보너스 | 힌트 여부와 무관하게 적용 |

**예시 A** — Medium, 2개 모두 찾음, 객체 1에 힌트 사용, 12초 남음:
- 객체 1: 5 × 0.5 = **2점** (소수점 내림)
- 객체 2: **5점** (힌트 미사용)
- 시간 보너스 (12초 남음, 80%): **+5점**
- 합계: **12점**

**예시 B** — Hard, 2개만 찾고 1개 찬스 소진, 시간 보너스 없음 (모두 못 찾음):
- 객체 1: **3점** (힌트 미사용)
- 객체 2: 3 × 0.5 = **1점** (힌트 사용, 소수점 내림)
- 객체 3: **0점** (찬스 소진)
- 시간 보너스: **+0점** (모두 찾지 못해 적용 안 됨)
- 합계: **4점**

### 요약

- 총 10문제, 최대 150점
- 난이도별 제한 시간: Easy=10초, Medium=15초, Hard=20초
- 시간 초과 시 미발견 객체 자동 오답 처리
- 목숨 시스템 없음 — 찬스 소진 또는 시간 초과 시 다음 문제로 진행
- 10문제를 모두 풀면 자동으로 결과 페이지로 이동

## 문제 데이터 구조

> ⚠️ **Breaking Change**: 기존의 `aiObjectName`, `hitbox`, `hint` 최상위 필드가 제거되고 `objects[]` 배열로 대체되었습니다.

각 문제는 다음 데이터를 포함합니다.

```json
{
  "id": 4,
  "image": "q4.jpg",
  "difficulty": "medium",
  "objects": [
    {
      "name": "고양이",
      "hitbox": { "x": 45.2, "y": 30.1, "width": 12.5, "height": 10.8 },
      "hint": {
        "type": "area",
        "value": { "cx": 50.0, "cy": 35.0, "radius": 20.0 }
      }
    },
    {
      "name": "우산",
      "hitbox": { "x": 20.0, "y": 60.0, "width": 8.0, "height": 12.0 },
      "hint": {
        "type": "text",
        "value": "하늘 아래 왼쪽을 보세요"
      }
    }
  ]
}
```

**제거된 필드:**
- `aiObjectName` → `objects[n].name`
- `hitbox` (최상위) → `objects[n].hitbox`
- `hint` (최상위) → `objects[n].hint`

**필드 설명:**
- `objects[n].hitbox`: AI 사물을 감싸는 판정 영역 (사각형). **이미지 대비 비율(%)로 저장**하여 화면 크기에 관계없이 동일하게 판정.
- `objects[n].hint`: 힌트 유형과 값. `area`는 원형 좌표(%), `text`는 문자열 (예: `"동물을 찾아보세요"`).
- `difficulty`: 난이도. Easy=objects 1개, Medium=objects 2개, Hard=objects 3개.

### 좌표 체계

모든 좌표는 **픽셀이 아닌 이미지 대비 백분율(%)**로 저장합니다. `x: 45.2`는 이미지 왼쪽 끝에서 45.2% 지점을 의미합니다. 이렇게 하면 데스크톱·모바일 등 화면 크기가 달라도 판정이 동일합니다.

### 판정 로직

플레이어가 클릭한 좌표를 이미지 기준 비율로 변환한 뒤, 히트박스 사각형 내부에 있는지 확인합니다.

```javascript
function isHit(clickX, clickY, hitbox) {
  return (
    clickX >= hitbox.x &&
    clickX <= hitbox.x + hitbox.width &&
    clickY >= hitbox.y &&
    clickY <= hitbox.y + hitbox.height
  );
}
```
